# Portfolio Website

A premium, high-performance, and responsive single-page portfolio website showcasing academic milestones, certifications, and technical projects.

## Tech Stack
- Frontend: React (v18)
- Build System: Vite
- CSS & Styling: Tailwind CSS, custom modern typography, and clean glassmorphic components
- Shaders: Three.js WebGL fragment shader background featuring flowing ribbons with soft glows

## Project Structure
- src/components/: Reusable design blocks and WebGL canvas components
- src/sections/: Modular sections of the portfolio (Hero, About, Projects, Education, Contact)
- src/styles/: Global stylesheets and style variable configurations
- public/: Publicly accessible static asset branding logos

## Build and Run Instructions

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation
Initialize and install required dependencies:
```bash
npm install
```

### Run Locally (Development Server)
Launch the development server:
```bash
npm run dev
```

### Compiling for Production
Build the client application files for deployment:
```bash
npm run build
```
The compiled output will be generated inside the `dist` directory.
